# GiveChain 

**GiveChain**은 블록체인을 활용한 투명하고 참여형 기부 플랫폼으로,

스마트컨트랙트를 통해 기부금의 흐름을 자동 기록하고, 국가별 / 분야별 랭킹 시스템으로 선의의 경쟁을 유도하는 Web3 서비스입니다.

# 💡 기능

1. 스마트컨트랙트 기반 기부 트래킹

- 기부금 송금 시 → 컨트랙트가 수신 주소(단체), 기부자 지갑 주소, 금액, 시간, 카테고리 자동 기록
- 모든 기부 기록이 블록체인에 저장되어, 누구나 언제든지 기부금의 흐름을 확인할 수 있습니다.<br><br>

2. 국가별 / 분야별 실시간 랭킹

- “국가별 기부 총액 TOP 10”, “환경 / 교육 / 보건 분야별 랭킹” 전 세계 누구나 블록 탐색기처럼 기부 데이터를 직접 검증 가능

- 투명한 선행 경쟁 구조 <br><br>

3. NFT or Soulbound Token 보상

- 기부 금액 / 횟수 / 영향력 기반 기부 인증 NFT 발급
- Bronze Donor / Silver Donor, 또는 특정 분야별 배지 (예: “Education Hero”)
- Soulbound Token(SBT)으로 만들어 양도 불가하게 구성 <br><br>

# 🛠️ 기술 구성
- Backend: Spring Boot, MySQL
- Frontend: Next.js, Ethers.js
- Blockchain: Solidity, IPFS
- DevOps: Docker, AWS, Vercel
